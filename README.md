# BLUEPRINT

**Что реализовано:**
1. Объявление переменных
    + Блок для объявления переменных 
    + Возможность объявлять несколько переменных через запятую
    + По умолчанию значение зависит от типа данных

2. Оператор присваивания
    + Блок для присваивания значения переменной
      
3. Базовые арифметические операции
    + Поддержка: `+` `-` `*` `/` `%`
    + В выражениях можно использовать переменные и константы

4. Оператор `If`
    + Блок для условия `>` `<` `==` `!=` `>=` `<=`

5. Реализация старта в Event start

6. Подсветка ошибок
    + Визуальное выделение блоков с синтаксическими ошибками

#### Категория «Я могу написать сортировку пузырьком»

1. Циклы
    + Реализация цикла `while` и `for`

2. Оператор `If-Else`
    + Реализован через Branch

3. Статические массивы
    + Объявление массивов
    + Обращение к элементам по индексу
    + Возможность использования в выражениях

4. Логические выражения
    + Поддержка операторов `AND` `OR` `NOT`

#### Категория «HARD»
1. Строковый и символьный типы данных  
2. Вещественные числа
3. Конвертация типов
4. Сохранение/загрузка алгоритмов
